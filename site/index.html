⚒ PoE2 Crafting Overlay

A transparent overlay for Path of Exile 2. Copy an item in-game and instantly see its mods, roll quality, what can still roll, the odds, and what currency to use next.
⬇ Download for Windows
Free & open source. Auto-updates. No account, no install hassle.

How to use

Run PoE2 inBorderless / Windowed Fullscreen Ctrl+C(in game) hover an item and copy it — the overlay updates automatically Ctrl+Alt+Oshow / hide the overlay Ctrl+Alt+Xclick-through (let clicks pass to the game)

Tip: enable Options → UI → Advanced Mod Descriptions in PoE2 so copied items include tier & prefix/suffix info.

What it shows

  • Every mod on the item, with roll quality (and whether a Divine Orb is worth it)
  • What can still roll — gated to your exact base — and the odds of hitting it
  • A searchable reference for every currency & omen
  • A suggested next crafting step

✓ Safe to use

It only reads your clipboard and draws a window. It never reads game memory, injects code, or automates input — the same approach as other GGG-tolerated tools. Mod data comes from the community RePoE2 project and refreshes automatically after patches.
